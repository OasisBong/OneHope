#
# 1. 넷드론 엔진의 유니티 패키지 설치하기
#
유니티 프로젝트를 생성한후 다운로드 받은 NetDroneUnity.unitypackage 를 실행하세요.
Importing Package에서 모든 파일을 Import 합니다.

이제 아래 순서와 같이 Build Settings에 Client/Scenes/* 를 추가해야 합니다.

* 0: ExampleMenu.unity
* 1: ExampleAutoConnect (NDP).unity
* 2: ExampleChat (NDP).unity
* 3: ExampleRoom (NDP).unity
* 4: ExampleWeb (HTTP).unity

Scene 순서와 동일한 Enum을 선언하여 사용하고있어 순서가 매우 중요합니다.

#
# 2. 서버 접속을 위한 기본값 설정
#
Client/Scenes/* 들의 Hierarchy를 살펴보면 Scripts 라는 GameObject를 발견할수 있습니다.

기본값에 대한 설명은 아래와 같습니다.

* Enable DNS: 고정 IP가 아닌 Dynamic DNS 사용자를 위한 옵션입니다. (고정 IP 사용자는 해당 옵션 끔)
* Remote Ip Or Domain: 서버 설치를 하지 않아도 테스트가 가능하도록 오리진 스튜디오에 설치(test.vogie.net)되어 있는 유니티 서버 예제입니다. (고정 IP 사용자는 IP 입력)
* Master Main Tcp: 서버에서 대기중인 Tcp Port 입니다.
* Master Sub Tcp: 서버에서 대기중인 Tcp Port 입니다.
* Slave Main Udp: 서버에서 대기중인 Udp Port 입니다.
* Enable Main Port: Main Port 설정 사용.
* Enable Reliable Udp: Reliable Udp 연결 사용.
* Default Login Id: 한글, 영문 모두 사용 가능합니다.
* Default Channel Id: 오리진 스튜디오에 설치되어 있는 유니티 서버 예제에서는 1 ~ 10 까지 사용이 가능합니다.

# 3. Tools 설치 및 사용 방법
#
# DataConverter
넷드론 엔진의 유니티 패키지에는 csv 파일을 bytes 포멧으로 일괄 변경하는 데이터 변환 도구가 포함되어 있습니다.
아래와 같이 Tools 경로 아래에 Tools/DataConverter.zip 파일을 복사하여 풀어주세요.

ROOT
- Assets
- ProjectSettings
- Tools (여기)

* Tools/Dataconverter/temp/* 아래에 테스트용 csv 파일이 들어있습니다.
* DataConverter.exe를 실행할 경우 지정된 위치의 모든 csv 파일들은 bytes 포멧으로 변경/저장 됩니다.
* 생성된 bytes 파일들은 Assets/NetDroneUnity/Client/Resources/Binary 아래에 복사하세요.

넷드론 엔진의 데이터 로더는 csv, bytes 포멧을 모두 지원합니다.
(csv의 구분자는 콤마(,)만 사용, 첫줄에 열이름이 반드시 필요)

* LoadBytes: Client/Scripts/DB/ChannelInfoDbm.cs 를 참고하세요.
* LoadCSV: Client/Scripts/DB/StageInfoDbm.cs 를 참고하세요.
