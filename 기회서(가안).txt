목표 : 스팀 그린라이트 등록

가제 : One Hope

게임 개요
섬에서의 사냥, 채집, 건축, 영역전 등을 통한 생존 및 키워드(퀘스트)를 통한 탈출

클라이언트 / standard alone 서버 / 인증용 웹서버

[시작화면]
클라이언트
- 계정 생성, 로그인 => 룸 리스트 / 생성, 조인

서버
- 웹 서버를 통한 인증(아이디 생성 또는 확인)
- Room Creat / Join

[게임 구성 요소]
 < UI >
- 메뉴 : 방 나가기, 옵션, 게임 종료
- 인벤토리 : 장비/상태창, 가방/재료
- 스킬창 : 테크 트리 형식
- 채팅창 : 유저 리스트 //치트키
- 지도창 : 지도
- 기본화면 구성 / 온리 스테미너 바
  
<맵>
- 섬 지형 / 오픈
- 동굴 / 해저동굴 같은 서브 맵(던전) 
- 오브젝트
  * 자원 / 돌, 나무, 줄기, 물 등
  * 동물 / 초식(1종), 육식(1종)
  * 적대 세력(몬스터(1종))
  * 아이템 상자(랜덤박스)
- 환경 : 낮/밤, 기상현상(눈, 비)

<캐릭터> : Account_Info, 
- 상태치 : 수치형 : HP, SP, 피로도, 배고픔, 갈증, 체온
	  상태형 : 중독(복통, 일시적 실명, 마비), 골(다리/팔), 출혈  
- 컨트롤러 : 키 컨트롤, 애니메이션 컨트롤

<NPC> : NPC_Info
- 상태치
- AI(컨트롤러)
- 드롭 아이템 목록

<스킬> : Skill_Info, 
- 채집 스킬 : 
- 건축 스킬 : 
- 제작 스킬 : 
- 요리 스킬 : 
- 사냥 스킬 : 

<아이템> : Item_Info / (휘발성 Table) RoomName_Itam_info
- 종류 : 재료용, 완성품, 강화용, 소모성, 이벤트성
- 아이템 수치 : 내구도(신선도 포함), 강도, 케릭터 상태치에 대한 적용 수치(예, 배고픔 -5)

<승리조건> : Victory_Info
- 탈출 : 탈출 요건 충족
	-> 배 제작 후 탈출 포인트 도달
	-> 비행기/헬리콥터 발견 시 제한 시간 내 구조요청(SOS, 불빛)
	-> 라디오 방송 수신 시 역송출(모스부호)
- 점령 : 위험 요소 완전 배제
	-> 환각 요소(저주받은 나무 등) 제거
	-> 원주민 제거

<이벤트>
- 비행기/헬기 등장
- 모스부호 관련 책자 발견
- 라디오 발견
- 원주민 발견
- 기타 등등
